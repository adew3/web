Структура сайту TicketGo
1. Головна сторінка
Головна сторінка містить основну інформацію про сервіс та форму пошуку квитків.
Елементи сторінки:
* Логотип TicketGo
* Меню навігації
* Форма пошуку квитків (звідки, куди, дата, пасажири)
* Кнопка «Знайти квитки»
* Популярні напрямки
* Блок «Як працює сервіс»
* Відгуки користувачів

2. Сторінка «Квитки»
На цій сторінці користувач може переглянути доступні рейси.
Функції сторінки:
* список доступних рейсів
* інформація про маршрут
* ціна квитка
* час відправлення та прибуття
* кнопка «Забронювати»
* фільтр по даті
* сортування по ціні

3. Сторінка «Мої бронювання»
Ця сторінка відображає всі квитки користувача.
Елементи:
* список активних бронювань
* історія поїздок
* деталі квитка
* можливість скасування бронювання

4. Сторінка «Профіль»
Містить інформацію про користувача.
Елементи:
* ім'я користувача
* email
* редагування профілю
* зміна пароля
* історія бронювань

5. Додаткові сторінки (Footer)
У нижній частині сайту знаходяться додаткові інформаційні сторінки.
Сторінки:
* Про нас
* Контакти
* Підтримка (FAQ)
* Політика конфіденційності
* Умови користування
